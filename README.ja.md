# VeryUsefulLibs (日本語)
## by taikun24
英語のREADMEは[こちら](README.MD)をご覧ください。
このフォルダには、さまざまな目的で使用できる非常に便利なライブラリが含まれています。
これらのライブラリは、タスクをより効率的に達成するのに役立ちます。
## 含まれるライブラリ
- **a.js**: とても便利な関数を提供するライブラリです。
- **b.js**: `a.js`に依存し、その機能を拡張するライブラリです。
- **c.js**: `a.js`と`b.js`の値を比較するライブラリです。
## 使用方法
これらのライブラリをプロジェクトで使用するには、次のようにrequireしてください。
```javascript
const a = require('./veryusefullibs/a');
const b = require('./veryusefullibs/b');
const c = require('./veryusefullibs/c');
```
