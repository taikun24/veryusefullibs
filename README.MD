## Libraries Included
- **a.js**: A library that provides a very useful function.

- **c.js**: A library that compares values of `a.js` and `b.js`.
## Usage
To use these libraries in your project, you can require them as follows:
```javascript
const a = require('./veryusefullibs/a');
const b = require('./veryusefullibs/b');
const c = require('./veryusefullibs/c');
```
